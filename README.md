# hand-gesture-led-control
A computer vision project using OpenCV and Python to control multiple LEDs based on finger detection and hand gestures.
# Features
- Detect fingers(thumb,index etc)
- Control LEDS based on finger input
- Real time hand tracking
# Technologies Used
- Python
- OpenCV
- MediaPipe
# Devices Used 
- Raspberry pi 3a+
- LEDs
- Breadboard
- Some Wires
# How it Works
- Camera captures hand
- Finger Detection algorithm runs
- LEDs turn ON/OFF based on gesture
# Future Improvements
- Add more gestures
- Control devices(fan,motor)
- Integrate with CSA(Client Server Architure) with ESP32 and Raspberry Pi
